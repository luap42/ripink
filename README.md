# ripink
A vanilla-js material-design ripple ink effect library

## Usage
### 1. Include the ripink.js
```html
<script src="/path/to/ripink.js"></script>
```
### 2. Commit the elements being affected by this library
```html
<script>
ripink.apply("css-selector", "css-color")
</script>
```
### 3. Enjoy!
